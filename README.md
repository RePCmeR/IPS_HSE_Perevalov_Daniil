# Игра-симулятор итеративной дилеммы заключенного

## Описание проекта
Проект представляет собой игру-симулятор, в которой два агента играют в дилемму заключенного какое-то конечное количество раундов. У них есть несколько вариантов действий, и им известны выигрыши в каждом из возможных исходов. Игровая составляющая заключается в том, что пользователь должен, исходя из стратегии оппонента, предложить такую стратегию, которая выигрывает в данной игре.

## Основные возможности (to be updated)
- Реализация собственной стратегии
- Возможность переключаться между раундами
- Различные турнирные сетки
- Возможность внедрения вероятностей в стратегии
- Уровни с несколькими игроками
